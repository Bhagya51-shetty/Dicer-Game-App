# 🎲 Dicer Game App

A colorful multiplayer Dice Game mobile application built using HTML, CSS, JavaScript and Apache Cordova.

## 🚀 Features
- Two Player Mode
- Score System
- Winning Logic
- Reset Game
- Sound Effects
- Smooth Animations
- Custom App Icon
- Splash Screen

## 🛠 Tech Stack
- HTML
- CSS
- JavaScript
- Apache Cordova

## 📱 Platform
- Android (APK)

## This app compatible with jdk 8 or 11(java) version .

## screenshots
- Dice1.png

## 📦 How to Run
1. Clone the repository
2. Run `cordova build android`
3. Install APK on device/emulator
